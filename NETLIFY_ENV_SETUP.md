# Netlify Environment Variables Setup

এখানে আপনার environment variables যোগ করুন Netlify ড্যাশবোর্ডে:

## Backend Environment Variables:
- FIREBASE_SERVICE_ACCOUNT
- FIREBASE_DATABASE_URL
- RETOOL_API_KEY
- RETOOL_DB_URL
- TELEGRAM_BOT_TOKEN
- TELEGRAM_ADMIN_CHAT_ID
- GEMINI_API_KEY
- TOROUTER_API_KEY
- APIVERVE_API_KEY
- OPENAI_API_KEY
- SESSION_SECRET
- PORT=3000
- NODE_ENV=production
- AVIATOR_API_KEY

## Frontend Environment Variables:
- VITE_AVIATOR_API_KEY

আরও বিস্তারিত তথ্যের জন্য `.env.example` ফাইল দেখুন।
